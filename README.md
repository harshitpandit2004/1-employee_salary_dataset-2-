# Employee Salary Prediction

##  Project Overview

This project predicts the annual salary of an employee using Machine Learning.

The model is trained using Multiple Linear Regression and deployed with Streamlit.

---

##  Features

- Predict Employee Salary
- Data Cleaning
- Feature Encoding
- Feature Scaling
- Machine Learning Model
- Streamlit Web App

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

---

## Run the Project

Install required libraries

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run app.py
```
